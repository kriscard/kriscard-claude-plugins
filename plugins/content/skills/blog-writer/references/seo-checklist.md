# SEO Checklist

## Before Writing
- [ ] Primary keyword identified
- [ ] Search intent understood
- [ ] Target word count set

## Title & Meta
- [ ] Title includes primary keyword (near start)
- [ ] Title under 60 characters
- [ ] Meta description 150-160 characters
- [ ] Meta includes keyword and call to action

## URL & Structure
- [ ] URL slug short with keyword
- [ ] URL uses hyphens (not underscores)
- [ ] No dates in URL

**Good:** `/blog/nextjs-app-router`
**Bad:** `/blog/2024/01/how-to-use-nextjs-app-router-guide`

## Content Structure
- [ ] Primary keyword in first paragraph
- [ ] H2 headings include secondary keywords
- [ ] Logical heading hierarchy (H1 → H2 → H3)
- [ ] Short paragraphs (2-4 sentences)
- [ ] Bulleted/numbered lists for scannable content

## Links
- [ ] 2-3 internal links
- [ ] 2-3 external links to authoritative sources
- [ ] Descriptive anchor text (not "click here")

## Images & Code
- [ ] All images have alt text
- [ ] Images optimized
- [ ] Code blocks specify language
- [ ] Code examples complete and runnable

## Final Checks
- [ ] Read aloud - flows naturally?
- [ ] Spell check passed
- [ ] All code tested
- [ ] Published date correct

## Keyword Placement

| Location | Required? |
|----------|-----------|
| Title | Yes |
| First paragraph | Yes |
| At least one H2 | Yes |
| URL slug | Yes |
| Meta description | Yes |
| Alt text | If natural |

## Don'ts
- Don't stuff keywords unnaturally
- Don't write for search engines over humans
- Don't ignore meta descriptions
- Don't publish thin content (<300 words for substantial topics)
