# Architecture Plugin

System design, technical leadership, and architecture documentation toolkit.

## Skills

### senior-architect
Comprehensive software architecture for designing scalable, maintainable systems.

**Tech Stack:** React, Next.js, Node.js, Express, React Native, Swift, Kotlin, Flutter, PostgreSQL, GraphQL, Go, Python

**Capabilities:**
- Architecture diagram generation
- System design patterns
- Tech stack decision frameworks
- Dependency analysis

### cto-advisor
Technical leadership guidance for architecture decisions, team scaling, and engineering strategy.

**Frameworks:**
- DORA Metrics (deployment frequency, lead time, MTTR, change failure rate)
- Team Topologies (stream-aligned, platform, enabling teams)
- ADRs (Architecture Decision Records)

### product-strategist
Strategic product leadership for vision setting, OKR alignment, and market analysis.

**Frameworks:**
- OKRs for goal alignment
- RICE for prioritization
- Jobs-to-be-Done
- North Star Metrics

## Commands

### /arch-doc
Generate architecture documentation with C4 diagrams and ADRs.

```bash
/arch-doc system           # Full system documentation
/arch-doc auth-service     # Specific service
/arch-doc "migrate to pg"  # Document a decision
```

### /check-spec
Validate specification quality for implementation readiness.

```bash
/check-spec docs/feature-spec.md
```

### /analyze-repo
Comprehensive repository analysis with Mermaid diagrams.

### /explain-codebase
Simple explanation of how the codebase functions and connects.

## Usage Examples

**Architecture design:**
```
Design a microservices architecture for an e-commerce platform
with user authentication, product catalog, and order management.
```

**Technical leadership:**
```
We're scaling from 5 to 20 engineers. What team structure
should we use? How do we manage tech debt during growth?
```

**Product strategy:**
```
Help me set OKRs for Q1. We want to focus on user retention
but also need to ship a major feature.
```
