# AI Development Plugin

LLM/RAG development and prompt engineering toolkit.

## Skills

### ai-engineer
LLM application and RAG system specialist.

**Expertise:**
- LLM integrations (OpenAI, Anthropic, local models)
- RAG pipelines and vector search
- Agent orchestration
- AI-powered feature development

### prompt-engineer
Expert prompt optimization for LLMs and AI systems.

**Techniques:**
- Chain-of-thought prompting
- Few-shot learning patterns
- Constitutional AI principles
- Production prompt strategies
